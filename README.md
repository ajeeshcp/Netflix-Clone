## Netflix clone

Netflix clone created by using React JS, TMDB API
## Run Locally on Your machine

Clone the project

```bash
  git clone https://github.com/ajeeshcp/Netflix-Clone.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
